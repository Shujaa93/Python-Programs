#  Rock Paper Scissors Game (Python)

##  Overview

This project is a simple command-line Rock, Paper, Scissors game developed using Python. The user plays against the computer, which makes random choices each round.

##  How It Works

The program asks the user to choose between rock, paper, or scissors by entering a short input (r/p/s). The computer randomly selects one of the options using Python’s random module.

Both choices are displayed using emojis for better visualization. The program then compares the user’s choice with the computer’s choice and determines the winner based on standard game rules.

The game continues in a loop until the user decides to exit.

##  Concept Used

* User input handling
* Conditional statements (if-else logic)
* Random selection using random module
* Loops for continuous gameplay
* Functions for structured code

##  Purpose

The purpose of this project is to practice basic Python concepts such as logic building, user interaction, and control flow through a fun and interactive game.

##  Summary

This is a beginner-friendly project that demonstrates how Python can be used to create simple interactive command-line games.
