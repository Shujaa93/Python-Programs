#  Number Guessing Game (Python)

A simple command-line game where the computer randomly selects a number between **1 and 100**, and you try to guess it.

---

##  Features

* Random number generation using Python's `random` module
* User input handling
* Helpful feedback: *Too high* / *Too low*
* Error handling for invalid input

---

##  How It Works

1. The program generates a random number between 1 and 100.
2. You enter your guess.
3. The program tells you if your guess is:

   * 📉 Too low
   * 📈 Too high
4. The game continues until you guess correctly.

---

## 💡 Example

```
Guess the number between 1 and 100: 50
Too low!
Guess the number between 1 and 100: 75
Too high!
Guess the number between 1 and 100: 63
Congratulations! You guessed the number.
```
